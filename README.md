# 🌟 Mumbai Guidebook Project 🌟
The Mumbai Guidebook project is a comprehensive online guide that provides users with information about the city of Mumbai, including its history, culture, and attractions. The project aims to be a one-stop destination for tourists and locals alike, offering a wide range of features and functionalities that make it easy to navigate and explore the city. The guide includes sections on hotels, food, shopping, and places to visit, making it an essential resource for anyone looking to experience the best of Mumbai.

## 🚀 Key Features
* 🏨 Hotel listings with features and amenities
* 🍴 Food listings with restaurant information and reviews
* 🛍️ Shopping listings with mall and market information
* 🗺️ Places to visit with descriptions and images
* 📝 User registration and login functionality
* 📊 Responsive design for a seamless user experience

## 🛠️ Tech Stack
* Frontend: HTML, CSS, JavaScript
* Framework: Bootstrap
* Icons: Font Awesome
* Authentication: Firebase JavaScript SDK
* Styling: Custom CSS (new.css, hotel.css, food.css, shop.css, style.css, login.css)

## 📦 Getting Started / Setup Instructions
### Prerequisites
* Node.js installed on your system
* A code editor or IDE of your choice
* A Firebase account for authentication

### Installation
1. Clone the repository using `git clone`
2. Install dependencies using `npm install`
3. Set up a Firebase project and enable the Firebase JavaScript SDK
4. Configure the Firebase SDK in your project

### Running locally
1. Open the project in your code editor or IDE
2. Run the project using a local server (e.g., `live-server`)
3. Access the project in your web browser at `http://localhost:8080`
